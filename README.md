# Votes service

Manage voting system.

- [Installation](#installation)
- [Commands](#commands)

## Installation

Set the database up.

```bash
make db-setup
```

## Commands

Run the API:

```bash
make run
```

Run tests:

```bash
make test
```

Connect to the database:

```bash
make db
```

Connect to the test database:

```bash
make db-test
```
